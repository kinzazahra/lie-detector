# 🕵️ Digital Lie Detector

A simple web application that analyzes whether a given statement is likely truthful or suspicious.

## 🔍 Features

* Sentiment analysis using TextBlob
* Typing speed/hesitation detection
* Keyword-based suspicion detection

## 🛠️ Tech Stack

* Python (Flask)
* HTML, CSS, JavaScript
* TextBlob

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000

## 📌 Note

This is a fun experimental project and not an actual lie detection system.
